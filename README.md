# Практическое задание по модуль 2.3
Консольное CRUD-приложение на Java с хранением данных в БД MYSQL с использованием ORM-фреймворка Hibernate

# Возможности

- Labels
  - Создание, обновление, удаление, просмотр
  - Автоматическая генерация ID
- Posts
  - Создание, обновление, soft удаление, просмотр, добавление Labels
  - Автоматическая генерация ID
- Writers
  - Создание, обновление, удаление, просмотр, добавление Posts
  - Автоматическая генерация ID

# Сборка проекта
- ./gradle build

# Запуск приложения
- необходимо поднять MySQL БД, указав проперти в DBProperties
  URL: jdbc:mysql://localhost:3307/testdb
  USERNAME: test
  PASSWORD: test
- java -cp build/classes/java/main kz.shyngys.AppRunner
- при первом запуске необходимо прогнать патчи миграции, вызвав пункт в меню
